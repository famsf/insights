Drupal Installation Instructions:
==================================

1. Download the Imagefield Zoom module from here: https://github.com/ujshah18/imagefield_zoom
2. Download the JQuery Zoom plugin from here: https://github.com/jackmoore/zoom (version: 1.7.15), unzip the file and
place the jquery.zoom.min.js file so that its path is: /libraries/jquery_zoom/js/jquery.zoom.min.js
3. Enable the module.

Module Usage:
==============

1. Add an image field to your content type/entity type.
2. On the manage display setting page, select the ImageZoom field formatter.
3. Click the gear on the right to configure how you want the Image zoom to work. You can learn more about each option
here: http://www.jacklmoore.com/zoom/
4. You'll see the zoom in action when the images are rendered on the node/entity pages.
